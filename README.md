# 🤖 AI Service — ESG Benchmark Comparator

This is the AI microservice built using Flask that generates ESG insights using the Groq LLM API.

It provides 3 endpoints:
- `/describe` → ESG description
- `/recommend` → ESG recommendations
- `/generate-report` → structured ESG report
